How to Set Up CodeAnywhere

I. Setting up FTP

  1. Go to CodeAnywhere.com and click the button labelled 'Sign up for Free'.
  2. Fill in your e-mail address, choose a password, and try to prove you're human.
  3. If you've convinced them that you're human, you should see the page below. Click the link to the editor, the one that says, 'here'.
  4. The connection wizard should be the next screen you see. You want to connect to your server using File Transfer Protocol, so click "FTP" in the left-hand column.
  5. For connection name, use your codename, all lowercase with no spaces.
  6. For hostname, type "trivium.phpwebhosting.com"
  7. For username, put in "slabweb" or "slabweb17", whichever you've been assigned.
  8. The password is not the password for CodeAnywhere or your e-mail. Use the class password written on the board.
  9. For 'Initial dir', put in "www/yourcodename", where 'yourcodename' is replaced with your actual codename.
  10. Doublecheck that it looks generally like the image below and then click 'Connect'.

II. Put your page on the Web

  1. Right-click on the connection (it will be labelled with your codename).
  2. Click on upload. Drag your files into the box that appears.
  3. Right-click on your firstpage.html and select 'Rename'. Rename the file to "index.html" .
  4. Go to your website ( eg. "slabweb17.phpwebhosting.com/yourcodename" ) and check that the files are in place. If you did not rename your firstpage to "index.html", you may see a page like the one below. Please fix this.
  5. You can also use the Project Checker to verify that your page is in the right place.
